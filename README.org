* 1. What's this?
The main purpose of this elisp is to install EMMS inside Spacemacs.

* 2. Install

1. Put this elisp into ~/.emacs.d/private/emms
2. Add emms to your .spacemacs file under dotspacemacs-configuration-layers
3. (Re)start Emacs!
